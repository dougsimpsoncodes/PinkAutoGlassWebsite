## PHASE 3: REVIEW GENERATION SYSTEM

### Step 3.1: Get Your Review Link

Once verified:
1. Go to Google Business Profile dashboard
2. Click "Get more reviews" or "Home" → "Get more reviews"
3. Copy your short review link

**Format:** `https://g.page/r/[YOUR_PLACE_ID]/review`

**Save this link - you'll need it!**

### Step 3.2: Update Website Component

Open file: `/Users/dougsimpson/Projects/pinkautoglasswebsite/src/components/GoogleReviewButton.tsx`

**Line 14:** Replace this:
```typescript
const GOOGLE_REVIEW_URL = 'https://search.google.com/local/writereview?placeid=YOUR_PLACE_ID';
```

With your actual review link:
```typescript
const GOOGLE_REVIEW_URL = 'https://g.page/r/YOUR_ACTUAL_PLACE_ID/review';
```
